## 🚀 Getting Started (3 Steps)

### Step 1: Run Locally (2 minutes)
```bash
npm install
npm run dev
```
Site is live at [http://localhost:3000](http://localhost:3000)

### Step 2: Customize (10 minutes)
Edit `app/page.tsx`:
- Update hero text (your own words)
- Update projects (your own work)
- Update capabilities (what you actually do)
- Change links in Navigation & Footer

### Step 3: Deploy (5 minutes)
```bash
npm run build    # Verify no errors
git push         # Push to GitHub
# Deploy on Vercel in 1 click
```

See `DEPLOYMENT_GUIDE.md` for detailed steps.
