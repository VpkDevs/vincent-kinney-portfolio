## KEY DESIGN DECISIONS

### Color System
- **Dark theme primary**: #0a0a0a (premium, not gray)
- **Accent**: #4d63ff (electric blue-violet, restrained but high-contrast)
- **Secondary accent**: #e0f2ff (subtle cyan, for interactive states)
- **Text**: #f5f5f5 (near-white, maximum readability)
- **Supporting text**: #a0a0a0 (gray, clear hierarchy)

**Why**: Dark theme feels premium and modern. Accent color is electric without being neon. Hierarchy is clear. No gradients, no glow, no excess.

### Typography
- **Display**: Geometric sans with personality (e.g., Inter, Recursive, Satoshi)
- **Body**: Clean, highly readable sans (Inter or system UI)
- **Emphasis**: Weight variation (600 instead of 400), size changes, never italic or all-caps
- **Line length**: 65 characters max for readability; break into columns on desktop

**Why**: Typography does the hierarchy work. Restrained but distinctive. No generic startup fonts.

### Layout
- **Max-width**: 1200px (premium spacing, not full-width)
- **Generous whitespace**: Spacing is premium signal
- **Single-column primary**: Projects, capabilities, philosophy stack vertically (not multi-column grids)
- **Mobile-first responsive**: Optimize for mobile, enhance for desktop

**Why**: Breathing room = premium. Content is the star, not the layout. Vertical stack feels editorial and intentional.

### Motion
- **Restrained**: Fade-ins on scroll, gentle hovers, no spin/bounce
- **Purposeful**: Animation communicates, doesn't distract
- **Duration**: 200-300ms for most interactions
- **GPU-accelerated**: Use transform/opacity only

**Why**: Smooth without feeling like a marketing site. Purposeful, not performative.

## CONVERSION STRATEGY

### Philosophy
- **No single "Hire Me" button everywhere** (it's annoying)
- **Intent-matched CTAs**: Different visitors need different calls
- **Multiple discovery paths**: Work → philosophy → lab → contact
- **Contact as outcome, not lever**: Forms support different conversations

### CTA Placement
1. **After hero**: Soft "View Featured Work" (scrolls to projects)
2. **After projects**: "Let's discuss" (contact form)
3. **After philosophy**: "Read the lab" (links to /lab)
4. **In footer**: Primary contact form

### Form Strategy
Three separate contact forms (not one generic form):
1. **Partnership / Co-Founder Interest** — For builders, collaborators, investors
2. **Product Feedback / User Inquiry** — For users of Shipwright, Narrative Engine, etc.
3. **General Inquiry** — For everything else

This signals selectivity and that different conversations matter.
