# 🚀 START HERE

Your complete portfolio website is ready. Here's what to do next.

## In 5 Minutes

1. **Open terminal** and navigate to this folder
2. **Run:**
   ```bash
   npm install
   npm run dev
   ```
3. **Visit** [http://localhost:3000](http://localhost:3000)
4. **You see your live site!**

## In 15 Minutes (Customize)

1. **Open** `app/page.tsx` in your code editor
2. **Find** the hero section (around line 160)
3. **Edit the text** to your own words
4. **Refresh browser** — changes appear instantly
5. **Keep editing:**
   - Projects section (line 200+)
   - Capabilities (line 290+)
   - Philosophy (line 310+)
   - Any other content

## In 1 Hour (Polish & Deploy)

1. **Update links** in `components/Navigation.tsx` and `components/Footer.tsx`
2. **Change colors** in `tailwind.config.js` if desired
3. **Add real images** to `public/images/` folder
4. **Test on mobile** (DevTools → Toggle Device Toolbar)
5. **Push to GitHub:**
   ```bash
   git init
   git add .
   git commit -m "My portfolio site"
   git remote add origin https://github.com/YOUR_USERNAME/vincent-kinney-portfolio.git
   git push -u origin main
   ```
6. **Deploy to Vercel:**
   - Go to [vercel.com](https://vercel.com)
   - Click "New Project"
   - Import your GitHub repo
   - Click "Deploy"
   - Your site is live!

## 🔧 Common Questions

**Q: Can I change the colors?**  
A: Yes! Edit `tailwind.config.js` — change `'accent': '#4d63ff'` to any color you want.

**Q: Can I add my own projects?**  
A: Yes! Edit `app/page.tsx`, find the `projects` array, add yours.

**Q: Can I use real images instead of emoji?**  
A: Yes! Add images to `public/images/`, update project `image` field.

**Q: How do I deploy?**  
A: Follow `DEPLOYMENT_GUIDE.md` — takes 5 minutes on Vercel.

**Q: Can the forms actually send emails?**  
A: Yes, but requires backend setup (see DEPLOYMENT_GUIDE.md).

**Q: Can I add a blog?**  
A: Yes, but comes later (see FILE_STRUCTURE.md future enhancements).

## 🛟 If You Get Stuck

**Issue: `npm install` fails**
```bash
# Delete node_modules and try again
rm -rf node_modules package-lock.json
npm install
```

**Issue: Port 3000 in use**
```bash
npm run dev -- -p 3001
```

**Issue: TypeScript errors**
- Check spelling in imports
- Verify file paths
- Run: `npx tsc --noEmit`

**Issue: Tailwind colors not working**
- Edit `tailwind.config.js`
- Restart dev server

**Issue: Can't deploy**
- Follow DEPLOYMENT_GUIDE.md step-by-step
- Double-check GitHub credentials
- Check Vercel dashboard for error messages
