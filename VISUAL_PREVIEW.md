## 🎨 COLOR SCHEME

```
Background (Primary):     #0a0a0a  (Very dark, premium black)
Background (Secondary):   #1a1a1a  (Slightly lighter)
Background (Tertiary):    #232323  (For contrast)

Text (Primary):           #f5f5f5  (Near white, high contrast)
Text (Secondary):         #a0a0a0  (Gray, supporting)
Text (Muted):            #666666  (Very subtle)

Accent (Primary):         #4d63ff  (Electric blue-violet)
Accent (Secondary):       #e0f2ff  (Subtle cyan for hover)
Accent (Hover):          #6a7bff  (Lighter blue)

Border:                   #333333  (Subtle dividers)
Success:                  Green
Error:                    Red

Effect: Premium, dark, not generic. High contrast for readability.
```

---

## ⚡ INTERACTION PATTERNS

**Buttons:**
- Outline style by default → Filled on hover
- Blue color (#4d63ff)
- 200ms smooth transition
- Readable text at all sizes

**Cards:**
- Subtle 1px border
- Hover: Border color changes to accent blue
- No shadow (clean design)
- Expandable for more content

**Links:**
- Blue color by default
- Underline on hover
- 200ms color transition
- External links show icon

**Forms:**
- Dark background (#1a1a1a)
- Blue border on focus
- Clear error states
- Success feedback (checkmark)

**Scroll:**
- Smooth scroll behavior
- Fade-in animations on scroll
- No janky parallax or distracting effects

## 📊 Typography Hierarchy

```
Hero Heading (h1)
56px / 1.2 line-height / bold
────────────────────

Section Title (h2)
40px / 1.25 line-height / bold
────────────────────

Subsection (h3)
28px / 1.3 line-height / bold
────────────────────

Body Text
16px / 1.6 line-height / regular
────────────────────

Supporting Text
14px / 1.5 line-height / regular
────────────────────

Emphasis: Use weight (600 instead of 400), not italics or all-caps
```
